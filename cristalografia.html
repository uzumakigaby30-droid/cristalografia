Guía de Cristalografía

Convención: a(u) frente, b(v) derecha, c(w) arriba — igual que Callister/Shackelford.

1. Ejes y Vectores

u → a   v → b   w → c
r = u·a + v·b + w·c

2. Nomenclatura

Punto
(x,y,z)

Posición fraccionaria. Centro BCC: (½,½,½).

[110] = 1·a + 1·b

Familia Dir.

Todas las equivalentes por simetría.

Plano

3. Trazar [uvw]

A — Origen (0,0,0). Índice negativo → mueve origen 1 unidad.
B — u·a → v·b → w·c
C — Flecha al punto final. Fracciones → ×MCD.

4. Planos (hkl)

1. Intercepts en a,b,c (∞=paralelo)   2. Recíprocos (∞→0)   3. Enteros ×MCD   4. Neg. con barra h̄

5. XRD — Ley de Bragg

nλ = 2d·sin(θ)
d = espaciado interplanar = a/√(h²+k²+l²) para cúbica
Picos aparecen donde se cumple la ley de Bragg para cada familia de planos.

El patrón XRD se calcula sumando la intensidad de Bragg de cada reflexión (hkl) ponderada por el factor de estructura y el factor de Lorentz-polarización.

Regla: [½,½,0]×2 = [110]. Miller siempre usa enteros.
